# AWS Nginx High Availability

Implemented a highly available Nginx web architecture on AWS as part of my cloud learning journey.

## Project Implementation

- Launched EC2 and installed Nginx
- Created a custom AMI after configuration
- Designed public and private subnet architecture
- Configured Application Load Balancer in public subnets
- Created Target Group with health checks
- Implemented Auto Scaling Group in private subnets
- Verified application access using Load Balancer DNS

## What I Learned

This project helped me understand:

- Load balancing concepts
- Auto scaling mechanisms
- Secure network isolation (public vs private subnets)
- High availability design in AWS
